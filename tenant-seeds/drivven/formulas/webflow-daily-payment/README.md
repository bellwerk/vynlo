# Drivven Webflow marketing payment

This private marketing formula is:

```text
((cash price × 1.15) ÷ 1.30 × 1.15 ÷ 24 ÷ 30)
```

It returns a daily marketing estimate rounded half-up to the nearest cent. For a cash price of 10,000 CAD, it displays 14.13 CAD/day.

It is not a contractual payment, lender quote, or Vynlo default. Webflow uses it only for Drivven listing presentation, and the formula version is stored with the mapped listing snapshot.
