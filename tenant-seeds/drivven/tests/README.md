# Drivven tenant tests

Tests are private and run only against synthetic/anonymized data.

Required suites:

- workspace-configuration package schema validation;
- stock and document sequence concurrency;
- Drive path/move/reconciliation;
- Webflow mapping and staging lifecycle;
- media derivatives;
- Drivven workflow transitions;
- RTB formula fixtures and invariants;
- PDF field coverage and visual regression;
- role/permission tests;
- migration dry run;
- tenant isolation when installed beside another synthetic workspace.
