# Release roadmap

## Release 1 — Vynlo core plus Drivven pilot

### Vynlo core

- tenancy, auth, MFA, RBAC, sessions, audit, approvals;
- mobile-first PWA and French/English architecture;
- inventory, costs, media, managed storage, VIN adapter, search;
- listing/provider framework;
- CRM with workflow-derived lead pipeline board/list, deals, trade-ins,
  third-party finance tracking, and one-time payments;
- document, calculation, tax-pack, workflow, numbering, custom-field, and export engines;
- API, outbox/worker, security, observability, operations, and testing;
- standard retail dealer starter pack;
- workspace onboarding and runtime configuration versioning.

### Drivven pilot

- import the Drivven seed into a runtime workspace;
- configure Auto BS Inc., Montreal, Sherbrooke, roles, stock numbering, and inventory workflow;
- Google Shared Drive and Webflow staging/production adapters;
- existing inventory migration;
- Drivven document and RTB development flow behind activation gates;
- mobile/desktop UAT and production hardening.

No separate repository or deployment is created for Drivven.

## Release 1.1

- richer safe admin configuration;
- improved import/reconciliation and tenant configuration export;
- notification preferences;
- expanded reporting and support tooling;
- retention/privacy request tooling;
- optional local screen lock;
- improved onboarding automation.

## Release 2

- additional storage/listing providers and approved tax packs;
- digital-signature provider framework;
- accounting integrations;
- lender API adapters;
- optional reconditioning workflow;
- controlled custom report/document builders.

## Future

- licensed market intelligence, appraisal, sourcing, and pricing recommendations;
- marketplace syndication and controlled automation;
- native mobile application;
- optional recurring payment servicing and collections;
- advanced automations and visual builders;
- service/parts/fleet modules only after validated demand.
